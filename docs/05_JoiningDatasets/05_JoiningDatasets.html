Joining data frames

Joining data frames

something_join(x, y)
  • left_join(): all rows from x
  • right_join(): all rows from y
  • full_join(): all rows from both x and y
  • semi_join(): all rows from x where there are matching values in y, keeping just columns from x
  • inner_join(): all rows from x where there are matching values in y, return all combination of multiple matches in the case of multiple matches
  • anti_join(): return all rows from x where there are not matching values in y, never duplicate rows of x
  • …

Joining on multiple variables

Set-up for multiple variables join

x_multi <- x %>% 
  mutate(id2=c(4,5,6)) %>% 
  select(id,id2,value_x)

x_multi
# A tibble: 3 × 3
     id   id2 value_x
  <dbl> <dbl> <chr>  
1     1     4 x1     
2     2     5 x2     
3     3     6 x3     
y_multi <- y %>% 
  mutate(id2=c(4,6,8)) %>% 
  select(id,id2,value_y)

y_multi
# A tibble: 3 × 3
     id   id2 value_y
  <dbl> <dbl> <chr>  
1     1     4 y1     
2     2     6 y2     
3     4     8 y4
