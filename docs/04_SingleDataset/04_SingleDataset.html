Select helpers

  • starts_with(): Starts with a prefix
  • ends_with(): Ends with a suffix
  • contains(): Contains a literal string
  • num_range(): Matches a numerical range like x01, x02, x03
  • one_of(): Matches variable names in a character vector
  • everything(): Matches all variables
  • last_col(): Select last variable, possibly with an offset
  • matches(): Matches a regular expression (a sequence of symbols/characters expressing a string/pattern to be searched for within text)

See help for any of these functions for more info, e.g. ?everything.

summarize and group_by

summarize for summary stats

  • Fun fact: summarize and summarise both work
  • Hadley Wickham, creator of the tidyverse is from New Zealand
  • mutate: observation level
  • summarize: aggregation of data
penguins %>% 
  summarize(avg_bill_len = mean(bill_len, na.rm = TRUE))
# A tibble: 1 × 1
  avg_bill_len
         <dbl>
1         43.9

💡Tip!

summarize() changes the data frame entirely. It collapses rows down to a single summary statistic, and removes all columns that are irrelevant to the calculation.

group_by for grouped operations

penguins %>% 
  group_by(species) %>% 
  summarize(avg_bill_len = mean(bill_len, na.rm = TRUE))
# A tibble: 3 × 2
  species   avg_bill_len
  <fct>            <dbl>
1 Adelie            38.8
2 Chinstrap         48.8
